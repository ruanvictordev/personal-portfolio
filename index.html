<Projetos/>

OuviP2

OuviP2 fez parte do meu Trabalho de Conclusão de Curso em Análise e Desenvolvimento de Sistemas. Este sistema propõe uma solução para a cidade de Pedro II, localizada no estado do Piauí, Brasil. Através desta plataforma web, pretende-se estabelecer uma forma eficiente e direta para os cidadãos reportarem questões urbanas e contribuírem para a melhoria contínua da cidade.

Sistema para Adoção

Projeto Web desenvolvido em HTML, Bootstrap, PHP e MySQL. Seu principal objetivo é divulgar animais de estimação disponíveis para adoção responsável. O projeto visa fornecer uma plataforma na web para que os usuários aprendam mais sobre a organização e apoiem seus esforços no bem-estar animal.

Proway Computers

O projeto “Proway Computers” é uma plataforma de e-commerce baseada em Angular e TypeScript que visa apresentar dispositivos eletrônicos para venda, simulando um ambiente de compra amigável.
